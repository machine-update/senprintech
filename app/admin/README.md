# SenPrinTech - Plateforme Premium de Personnalisation

## 🚀 Stack Technique
- **Framework**: Next.js 14 (App Router)
- **Langage**: TypeScript
- **Style**: Tailwind CSS + Shadcn UI
- **Database**: PostgreSQL + Prisma ORM
- **Auth**: NextAuth.js (RBAC: USER, BUSINESS, ADMIN)
- **State**: Zustand (Panier & Designer)
- **Paiement**: Stripe (Prêt pour intégration)
